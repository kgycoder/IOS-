# iOS Style Android Launcher

Compose 기반 Android 12+ 커스텀 Launcher 프로젝트입니다.

## Modules
- `app`: Launcher entry and integration
- `core-ui`: theme + glass components
- `feature-home`: iOS style home grid/dock/pages
- `feature-widget`: smart widget stack
- `feature-controlcenter`: control center panel
- `feature-player`: now playing UI
- `feature-applibrary`: app library + search

## Build
```bash
gradle :app:assembleRelease
```
